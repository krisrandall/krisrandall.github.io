What is datasync.js for?

datasync.js is for creating a web-client/mobile app which is able to fetch data from, and store data to, a web server.

There are obviously 2 components, a server DB and the client App.




Setting up the Server

  1. The first step is to setup your (MySQL) database on the server
  2. Next put the datasync-server direcotry (PHP code) onto the server
  3. Set up the config for your DB and your client App in datasync-server/config.php
That's all that needs to be done on the server!

Setting up the Client

  1. Add the datasync-client directory to your client app
  2. On the server run datasync-server/generateClientJS.php, and save the generated code as datasync-client/mydb.js
  3. Delete datasync-server/generateClientJS.php if you don't want everyone on the internet to be able to see your data structure
Now your client is setup!

You can use datasync in your code: 
					
	datasync.fetchFromServer(table, where, success, fail);
	datasync.storeToServer(table, where, success, fail);
